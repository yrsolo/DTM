# DTM GitHub Pages showcase

Static public presentation for DTM.

- Entry: `site/index.html`
- Styles: `site/styles.css`
- HLS video player and reveal behavior: `site/app.js`
- Generated architecture art: `site/assets/`
- Reproducible backplate prompts: `site/diagram-prompts.md`
- Deployment: `.github/workflows/pages.yml`

Exact architecture labels and flows live in deterministic SVG files. Generated images are used only as low-opacity backplates.
