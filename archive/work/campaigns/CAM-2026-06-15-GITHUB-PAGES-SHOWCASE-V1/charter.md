# CAM-2026-06-15-GITHUB-PAGES-SHOWCASE-V1 Charter

## Problem
- DTM has no concise public presentation surface suitable for a resume or an engineering portfolio.

## Goal
- Publish-ready GitHub Pages landing page that presents DTM as a serious design-operations system, links backend and frontend repositories, embeds the promo video, and explains the system architecture.

## Non-goals
- Changing backend or frontend product behavior.
- Publishing or merging to `main`.

## Exit Criteria
- Static showcase page and generated visual assets exist in the repository.
- GitHub Pages deployment workflow exists.
- Page is verified locally at desktop and mobile widths.
- Architecture claims are checked against current backend and frontend runnable artifacts.
