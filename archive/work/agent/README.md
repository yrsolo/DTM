# Agent Archive

Historical agent harnesses and retired operator-side prompts live here once they no longer describe the active runtime contour.

## Structure

- `legacy_smokes/` - smoke scripts tied to removed `src.legacy`, `src.services`, or other retired runtime shapes

## Rule

- Active helper scripts stay in `agent/` only if they import current paths and still validate the live system.
- Historical harnesses move here instead of lingering in the active `agent/` root.
