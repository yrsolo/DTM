# Contributing

- System documentation goes to `docs/`.
- Delivery process, campaigns, and execution tracking go to `work/`.
- New planned campaigns should be created in `work/roadmap/campaigns/`.
- Active execution should be tracked in `work/now/`.
- Completed campaigns should be summarized in `work/archive/campaigns/<CAM>/closeout.md`.
- Keep changes small, reversible, and aligned with `AGENTS.md`.
